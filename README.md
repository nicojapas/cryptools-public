# Cryptools

Cryptools is a web application that provides various tools and information related to cryptocurrencies and blockchain technology. The app allows users to access the latest news, see newly created contracts on the Binance blockchain, test tokens to see if they are safe to buy, and much more.

Features
Some of the features of the Cryptools webapp are:

Latest news: The app displays the latest news related to cryptocurrencies from various sources.
Binance contracts: The app shows newly created contracts on the Binance blockchain.
Token testing: The app provides tools to test tokens and determine whether they are safe to buy.
Crypto trends: The app displays the latest trends in the crypto space, including prices, market capitalizations, and more.
Easy-to-use interface: The app is designed to be user-friendly and easy to navigate.
Installation
To run Cryptools on your local machine, follow these steps:

Clone the repository: git clone https://github.com/nicojapas/cryptools.git
Navigate to the project directory: cd cryptools
Install dependencies: yarn install
Start the server: yarn start
Open the app in your browser at http://localhost:1234
